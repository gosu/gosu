Gosu is a 2D game development library for the Ruby and C++ programming languages, available for Mac OS X, Windows and Linux.
MSVC++ 2005 or 2008 needed for the Windows C++ version, but the free Express versions are fine.

All languages have access to:

  • 2D graphics and text, accelerated by 3D hardware
  • Sound samples and music in various formats
  • Keyboard, mouse and gamepad/joystick input
Ruby/Gosu also features:
  • Translations of the Ruby tutorial into Traditional Chinese, Spanish and French, as well as Polish material—see DocsOverview in the Wiki
  • Installation via gem install gosu, complete with examples
  • RMagick integration, shown in an example game
  • A tutorial on how to use the Chipmunk physics library with Gosu: RubyChipmunkIntegration, written by Dirk Johnson
  • An example for using ruby-opengl to add rich 3D effects to 2D Gosu games

What's next?

Current work focuses on bringing Gosu to the iPhone, making it easy to use it with Ruby 1.9 on all platforms, and making it easier to integrate robust custom OpenGL.
